# Assets

This directory should contain the following app icons:

- icon.png - Main app icon (1024x1024)
- adaptive-icon.png - Android adaptive icon (1024x1024)
- splash.png - Splash screen image (1284x2778)
- favicon.png - Web favicon (16x16 or 32x32)

For development, Expo will use default placeholders if these files are missing.
